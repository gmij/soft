# 飞书

飞书（Lark）是字节跳动推出的企业协作与管理平台，整合即时通讯、日历、在线文档、云盘、工作台等功能，为企业提供高效的协作体验，帮助团队提升工作效率。

## 功能特点

- 即时通讯：支持单聊、群聊、音视频通话
- 飞书文档：在线协作编辑文档、表格、思维笔记
- 视频会议：高清视频会议，支持妙记实时转写
- 日历和会议室：智能日程管理，会议室预订
- 飞书OKR：目标管理和绩效跟踪
- 飞书审批：灵活的审批流程配置
- 飞书云盘：企业级文件存储和分享
- 机器人和应用：丰富的机器人和第三方应用生态

## 系统要求

- Windows 7 或更高版本
- macOS 10.13 或更高版本
- Linux（Ubuntu 18.04 或更高版本）
- iOS 12.0 或更高版本
- Android 6.0 或更高版本

## 安装说明

1. 访问官方网站下载对应系统的安装包
2. 运行安装程序
3. 使用手机号或邮箱注册登录
4. 加入企业组织或创建团队

## 使用说明

飞书采用全平台同步设计，支持在手机、电脑、平板、网页等多端使用。其文档功能支持多人实时协作，所有修改自动保存。

## 为什么推荐这个软件？

**Q: 飞书主要解决什么问题？**

A: 飞书致力于打破信息孤岛，提供一站式的协作平台。它将沟通、协作、知识管理整合在一起，减少在不同工具间切换的时间成本。特别是其在线文档和视频会议功能，能够大幅提升团队协作效率。

**Q: 相比其他企业协作软件有什么特色？**

A: 飞书的特色在于产品设计的连贯性和协作的流畅性。比如在文档中可以直接@同事、创建会议、发起审批；视频会议中的"妙记"功能可以实时转写并自动生成会议纪要。飞书的设计理念是让工具更好地服务于人，而不是让人去适应工具。

**Q: 适合哪些团队使用？**

A: 飞书特别适合注重效率、崇尚协作文化的现代化团队，包括互联网公司、创新型企业、远程团队等。其先进的产品理念和优秀的用户体验，使其成为追求高效协作的团队首选。

**Q: 飞书文档有什么优势？**

A: 飞书文档支持多人实时协作编辑，所有修改自动保存，无需手动保存。支持多种内容类型混排（文字、表格、代码块、思维导图等），并且可以在文档内直接@同事、插入文件、创建日程等，实现了真正的"All in One"协作体验。

**Q: 飞书的定价如何？**

A: 飞书提供免费版本，包含核心的即时通讯、文档协作、视频会议等功能。对于需要更多高级功能（如更大存储空间、更长会议时长、高级管理功能）的企业，可以选择商业版或企业版。
