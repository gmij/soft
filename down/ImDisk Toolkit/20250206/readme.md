# ImDisk Toolkit - 20250206

ImDisk Toolkit 是一个功能强大的 Windows 虚拟磁盘工具套件，集成了 ImDisk Virtual Disk Driver、DiscUtils 库和现代化的图形用户界面。

## 主要功能

- **虚拟磁盘创建**：支持创建虚拟硬盘、软盘或 CD/DVD 驱动器
- **RAM 磁盘**：使用系统内存创建高速 RAM Disk
- **映像文件挂载**：支持挂载多种格式的镜像文件
- **系统集成**：在 Windows 资源管理器中右键菜单快速挂载
- **自动挂载**：支持系统启动时自动创建和挂载虚拟磁盘

## 系统要求

- Windows 7/8/8.1/10/11 (64-bit)
- Windows Server 2008 R2 及以上版本

## 安装说明

1. 解压 ImDiskTk-x64.zip 文件
2. 运行安装程序
3. 按照向导完成安装

## 注意事项

本工具适用于需要虚拟磁盘功能的用户，建议在安装前关闭杀毒软件以避免误报。
