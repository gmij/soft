# ImDisk Toolkit

ImDisk Toolkit 是一款轻量级但功能强大的虚拟磁盘管理工具，支持创建内存磁盘、虚拟硬盘和挂载镜像文件，帮助用户高效管理存储资源。

## 功能特点

- **创建虚拟磁盘**：支持在内存中创建高速虚拟磁盘，用于缓存或临时存储。
- **挂载镜像文件**：支持挂载 ISO、IMG 等多种镜像文件，无需解压直接访问内容。
- **管理虚拟硬盘**：支持动态调整虚拟硬盘大小，提供灵活的存储管理方案。

## 系统要求

- Windows 7 或更高版本（包括 Windows 10 和 Windows 11）
- 适用于 64 位系统

## 安装说明

1. 下载软件包：直接下载 [ImDisk Toolkit 版本 20250206](https://example.com/download/ImDiskTk-x64.zip)。
2. 解压缩下载的文件 `ImDiskTk-x64.zip`。
3. 双击运行安装程序，按照安装向导提示完成安装。

## 使用说明

1. 打开 ImDisk Toolkit 软件。
2. 在主界面选择所需功能，例如创建内存磁盘或挂载镜像文件。
3. 根据提示输入相关参数，例如磁盘大小、镜像路径等。
4. 点击“确定”完成操作，开始使用虚拟磁盘或挂载的文件。
