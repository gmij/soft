# Ventoy

Ventoy is an open-source tool that allows users to create bootable USB drives for ISO/WIM/IMG/VHD(x)/EFI files. Instead of formatting the USB drive repeatedly, you can simply copy and paste files onto the drive, making it a highly efficient and user-friendly solution for bootable media creation.

## Features

- **Multi-Boot Support**: Create a single USB drive capable of booting multiple operating systems.
- **No Reformat Needed**: Add or remove ISO files without reformatting your USB drive.
- **Wide Compatibility**: Supports Legacy BIOS and UEFI modes, along with over 1,000 tested ISO files.

## System Requirements

- **Windows**: Windows 7, 8, 8.1, 10, or later
- **Linux**: Most major Linux distributions (Ubuntu 16.04+ recommended)
- **USB Drive**: Requires a USB drive with at least 8GB of storage

## Installation

1. Download the latest version of Ventoy for your operating system:
   - Version **v1.0.93**: [Download Ventoy-1.0.93.zip](https://www.ventoy.net)
2. Extract the downloaded ZIP file.
3. Run the **Ventoy2Disk** executable for your OS.
4. Select your USB drive, click "Install," and confirm the operation.
5. Once installed, copy ISO files directly onto the USB drive.

## Usage

1. Copy your desired ISO/WIM/IMG/VHD(x)/EFI files directly to the USB drive.
2. Plug the USB drive into your target computer.
3. Boot from the USB drive and select the desired ISO file from the menu.
4. The selected operating system will boot up automatically.
