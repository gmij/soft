# Ventoy

Ventoy 是一款开源的多启动工具，能够帮助用户轻松创建可启动的 USB 设备。只需将 ISO 文件拷贝到 U盘中，无需格式化即可直接启动，支持多种操作系统和工具镜像。

## 功能特点

- **多启动支持**：支持在一个 USB 设备上存储和启动多个 ISO 文件。
- **无格式化操作**：无需每次写入镜像时格式化 U盘，提升使用效率。
- **广泛兼容性**：支持超过 900 种 ISO 文件格式，包括 Windows、Linux 和各种工具镜像。

## 系统要求

- Windows 10 或更高版本
- macOS 10.15 或更高版本
- Linux（Ubuntu 20.04 或更高版本）

## 安装说明

1. 下载 Ventoy 的安装包 [Ventoy-1.0.93.zip](https://example.com/download/Ventoy-1.0.93.zip)。
2. 解压缩安装包。
3. 运行安装程序，根据提示选择目标 USB 设备并完成安装。

## 使用说明

1. 将 ISO 文件拷贝到已安装 Ventoy 的 USB 设备中。
2. 插入设备并启动计算机。
3. 在启动菜单中选择 Ventoy，并选择要启动的 ISO 文件即可。
