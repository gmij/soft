# WinRAR

WinRAR 是一款强大的压缩管理工具，支持 RAR 和 ZIP 文件的创建与解压，兼容多种压缩格式，帮助用户高效管理文件并节省存储空间。本版本为 5.01 烈火版，提供增强的性能与稳定性。

## 功能特点

- **多格式支持**：支持 RAR、ZIP、CAB、ARJ、LZH、TAR、GZ、7Z 等主流压缩格式。
- **分卷压缩**：方便将大文件分割成多个小文件，便于存储与传输。
- **加密功能**：提供 AES-256 位加密，保障文件的安全性。
- **快速解压缩**：优化的算法提升了解压与压缩速度。
- **界面友好**：简单直观的用户界面，易于使用。

## 系统要求

- **Windows**: Windows 10 或更高版本
- **macOS**: 目前未提供官方版本（可使用类似工具如 The Unarchiver 替代）
- **Linux**: 需使用命令行工具 UnRAR（支持 Ubuntu 20.04 或更高版本）

## 安装说明

1. 下载 WinRAR 5.01 烈火版的安装包（文件名：WinRAR.zip）。
2. 解压下载的压缩包。
3. 双击运行安装程序，并按照提示完成安装。
4. 安装完成后即可直接使用。

## 使用说明

1. 打开 WinRAR 软件。
2. **压缩文件**：选择需要压缩的文件或文件夹，右键点击并选择“添加到压缩文件”。
3. **解压文件**：右键点击需要解压的压缩文件，选择“解压到当前文件夹”或指定路径。
4. **分卷压缩**：在压缩选项中设置分卷大小即可。
5. **加密文件**：勾选“设置密码”选项，并输入加密密码。
